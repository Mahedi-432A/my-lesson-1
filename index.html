We are learning basic CSS

CSS means Cascading style sheet

We can apply CSS in 3 ways :

  1. inline
  2. internal or embedded
  3. external

inline CSS is used in the line of html tags, that means we write css in the line where we write html language.

Internal CSS

Here we write CSS internally in the html file. We write style tag inside head and then we write all styles in the style tag. Here we need selector to apply style, for example:

  • h1 p img etc, like any tags
  • id or class declared in a tag, then we call the id or tag and apply CSS

For external CSS we create a CSS file and use selector to apply CSS. Then we connect the CSS file to the html file.

Selectors

there are some type of selector. tags is used like a selector , ID and Class also used as selector
